Creates a "video QR-code" to share data between smartphones. Is intended as a proove of concept as well as a test for maximum data transfer rates.
It was inspired by the fact that an iPhone camera has a much higher data transfer rate then a bluethooth receiver, but only the latter is used for 
data sharing like AirDrop. The final product (non-Python version) should enable App users to share a file/Video/Link... by converting it into a repeating sequence of
RGB-Qr-codes, which can be read from the senders screen by users with the corresponding App.
